# Broadcast News NL Mini (N-Best 2008)

| | |
|---|---|
| **Domain** | broadcast news |
| **Language** | nl |
| **Size** | 0.77 hours |
| **License** | restricted |

3-file subset of N-Best 2008 BN-NL for smoke-testing. Files 001 (NOS 8-uur Journaal), 012 (NOS 8-uur Journaal), 014 (BNR Nieuwsradio) — ~46 minutes total. File 014 contains challenging F2 (telephone/interview) speech from Alexander Pechtold. Note: 3-file subset is not representative of the full BN-NL score due to high proportion of difficult F2 material.

**Reference:** Van Leeuwen, D.A., Kessens, J., Sanders, E. & Van den Heuvel, H. (2008). Results of the N-Best 2008 Dutch Speech Recognition Evaluation. https://citeseerx.ist.psu.edu/document?repid=rep1&type=pdf&doi=32b10cb0f4cb99ba934f5be5066638a5ad9b19f2
